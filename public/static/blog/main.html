Masalah utamanya terletak pada definisi CSS untuk **Light Theme**. Pada bagian `body.dark .glass-panel`, Anda menyertakan properti `backdrop-filter: blur(16px);`. Namun, pada bagian `body.light .glass-panel`, properti tersebut **hilang**, sehingga panel menjadi transparan tetapi tidak nge-blur. Berikut adalah perbaikan kode lengkapnya.
